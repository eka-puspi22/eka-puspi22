### 🔄 Cerita Career Switch Saya

Saya berlatar belakang pendidikan **Ilmu Nutrisi dan Teknologi Pakan**, dan
kini sedang bertransisi karier menuju dunia **Data Analyst**, dengan rencana
jangka panjang untuk berkembang ke arah **Data Scientist**.

Ketertarikan saya pada data tumbuh dari pengalaman semasa kuliah dan bekerja
di bidang nutrisi & pakan, di mana banyak keputusan diambil berdasarkan data
dan hasil pengujian. Dari situ saya sadar bahwa kemampuan mengolah data dan
menerjemahkannya menjadi insight adalah skill yang ingin saya kuasai lebih
serius.

Saat ini saya sedang mengikuti **Data Analyst Bootcamp di @dibimbing.id**
untuk membangun fondasi yang kuat sebelum melangkah lebih jauh ke Data
Science. 🚀
